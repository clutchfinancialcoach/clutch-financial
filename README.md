# Clutch Financial

Professional website for Clutch Financial, a financial coaching and education brand helping professionals and growing families build strong habits, make confident financial decisions, and follow through.

## Project Status

The project is in Phase 1: professional website. The current site includes Home, About, Services, FAQ, and Contact pages.

## Documentation

Start here:

- `PROJECT_BRIEF.md`: project goals, audience, positioning, and scope.
- `BRAND.md`: mission, vision, ideal clients, voice, visual identity, and phrases to avoid.
- `ROADMAP.md`: phased plan from professional website through future resources and client tools.
- `COPY.md`: approved messaging, CTA language, page copy, FAQ language, and disclaimer.

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS

## Local Development

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open `http://localhost:3000` in your browser.

## Quality Checks

Run linting:

```bash
npm run lint
```

Create a production build:

```bash
npm run build
```
